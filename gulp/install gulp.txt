	инициализация проекта: создание манифеста - package.json
3 npm init

	//npm i - установить все зависимости если надо и если нет папки node_modules в проекте
	//после можно папку node_modules удалить и после через npm i восстановить уже с новыми связями если нужно обновить папку

5 создаем общую структуру проекта, а именно
	dist - продакшен (готовый продукт после сборки)
	app - папка разработчика - все исходные материала проекта (оригинальные)
		js,img,less,fonts,css, index.html	

6 создаем в корне проекта gulpfile.js для написание на gulp
7 формируем данный файл

	var ...

	//--save-dev обязательно для дозаписи в package.json
